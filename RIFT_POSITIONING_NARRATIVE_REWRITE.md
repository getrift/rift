# Rift Positioning + Narrative Rewrite

Date: 2026-06-02  
Scope: current `/projects/rift` website positioning for the private-beta launch

## Verdict

The sharper positioning is:

> **Make your AI work compound.**

Keep it. It is active, short, and outcome-led. The important change is not the H1 anymore; it is the proof order underneath it. The page must prove source-backed recall before it proves agent reuse.

## Product Truth To Preserve

- Rift imports export files from ChatGPT, Claude, Grok, and Gemini.
- Rift captures new Claude Code and Codex sessions after install/configuration.
- First import and keyword search work without Codex, embeddings, or an AI worker.
- Codex and embeddings are optional enrichment paths, not prerequisites for first recall.
- Connected tools do not magically receive context; they can ask Rift for source-backed context after setup.
- The supported setup path today is the terminal installer.
- A double-click Mac package is planned for beta.19, but it is not the live path yet.

## Hero Copy To Ship

Eyebrow:

> Private beta for Mac users with AI exports

H1:

> Make your AI work compound.

Subhead:

> Rift turns ChatGPT, Claude, Grok, and Gemini exports into a searchable local archive on your Mac. Find the answer, decision, or constraint you already worked out, with the original conversation attached. Connected tools can ask Rift for that context later.

Primary CTA:

> Start beta setup

Secondary CTA:

> See how it works

## Narrative Spine

1. Promise: AI work should accumulate value instead of resetting across tools.
2. First proof: import exports and search locally with sources.
3. Mechanism: Rift parses the archive, keeps source conversations visible, and makes first recall possible without Codex or embeddings.
4. Beta honesty: terminal installer today; Mac package planned for beta.19.
5. Second-act value: connected tools can ask Rift for the same context later.

## Page Sequence

1. Nav
2. Hero
3. Source-backed search proof
4. How it works
5. Beta status
6. Agent reuse proof
7. Final CTA
8. Footer

This is intentionally lean. Private beta does not need a mature ARR-style homepage. It needs a truthful page that high-fit users can trust.

## Copy Rules

Use:

- searchable local archive
- exports
- source-backed search
- original conversation attached
- connected tools can ask Rift
- first import/search works without Codex
- terminal installer today
- Mac package planned for beta.19

Avoid:

- broad "every model/tool" claims
- automatic account-history ingestion language
- vague infrastructure jargon in the hero
- implying Codex or embeddings are required for first value
- implying the Mac package is validated or available today
- making the terminal demo the first proof

## Final CTA

Headline:

> Search the work already in your AI exports.

Body:

> Private beta seats are opening for Mac users with real ChatGPT, Claude, Grok, or Gemini exports to test. Start with local import and source-backed search; connect agents after the archive proves useful.
